Сквозная аналитика для ПАО «Аэрофлот»

  • Единая модель получения и обработки данных из разнородных источников
  • Оцифровка сквозного процесса продажи билетов
  • Сокращение сроков подготовки отчетности

Предпосылки проекта

Сотрудники вручную контролировали показатели операционной обстановки по выполнению пассажирских рейсов, существующие системы учитывали взаимовлияние факторов частично. Это приводило к повышенным затратам ресурсов, задержкам рейсов.

Цели проекта

Централизация и автоматизация обработки постоянно меняющихся факторов - отложенные дефекты, характер перевозимого багажа, погода, квалификация экипажа
Повышение скорости принятия оперативных решений, например, замена воздушного судна
Снижение задержек в исполнении рейсов, оптимизация использования воздушного парка авиакомпании

Уникальность проекта

  • Интеграция со множеством производственных систем, обеспечение обработки разнородной информации в online режиме
  • Создание модели рейса с учетом разнородных факторов: характеристики воздушного судна, пассажиры, багаж, грузы, возможности аэропортов
  • Глобальность проекта. Автоматизировано около 4 000 рабочих мест сотрудников производственных подразделений, представителей авиакомпании в аэропортах, бортпроводников, пилотов

Ценность для клиента

Внедрение системы контроля операционных ограничений позволило:

  • Получать полную информацию о планируемом рейсе из множества производственных систем
  • Автоматически учитывать взаимосвязи разных факторов выполняемого рейса на основе настраиваемых правил
  • Заранее формировать операционные ограничения по конкретному рейсу
  • Оперативно информировать персонал о необходимости внесения корректировок в планирование рейсов
  • Снизить количество ошибок персонала, выполняющего ручной контроль таких факторов
